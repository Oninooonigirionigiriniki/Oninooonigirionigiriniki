## コミットメッセージの書き方
### テンプレート
```
[種類] (コミット要約)
```
### コミット種別
| 種類 | 説明 |
| :-- | --- |
| `Add` | 新規機能・ファイルの追加 |
| `Update` | 機能修正・変更 |
| `Fix` | バグ修正 |
| `Remove` | 機能・ファイル削除 |
| `Docs` | ドキュメント修正 |

### 例
```
[Add] ユーザー登録機能の追加
```
